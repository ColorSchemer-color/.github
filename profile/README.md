# ColorSchemer — Portable Color Scheme & Contrast Tool

## About ColorSchemer

ColorSchemer provides a complete color scheme and contrast checking environment for UI/UX designers who need harmonious palettes. ColorSchemer combines color wheel, color harmonies, contrast checker, color picker, and export. ColorSchemer enables users to generate monochromatic, complementary, triadic, tetradic, and analogous schemes. ColorSchemer runs on desktop platforms. ColorSchemer is trusted by UI designers, web designers, graphic designers, and accessibility specialists worldwide.

## Deep Dive — Color Scheme & Contrast Notes

Color wheel: hue, saturation, brightness, RGB/HEX/HSL values
Harmonies: monochromatic, complementary, triadic, tetradic, analogous
Contrast checker: WCAG 2.1, AA/AAA ratings, foreground/background
Color picker: screen magnifier, sample any color, copy HEX/RGB
Export: CSS variables, Swift UIColor, Android XML, PNG palette
